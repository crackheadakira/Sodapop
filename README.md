# Sodapop, a local file music player with a sleek modern UI. 
## This is a work in progress, so there may be bugs. If you do find any, please report them in the Issues tab.

<p align="center"><img src="https://i.imgur.com/SjYdnLq.png" width="100%"/></p>

## Features
- Play local audio files, currently we only support mp3 and flac files, but this can easily be expanded in the near future.
- Playlists, you can create your own playlists and add songs to them.
- Discord Rich Presence, you can show what you're listening to on Discord.
